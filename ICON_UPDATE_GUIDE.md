# Инструкция по обновлению иконки приложения StatApp

## Что мы сделали

Заменили стандартную иконку-привидение на ваш логотип StatApp во всех необходимых местах.

## Созданные файлы изображений

В папке `public/` теперь есть следующие изображения:

1. **icon.png** - основная иконка (512x512px)
2. **icon-200x200.png** - иконка для Farcaster манифеста (200x200px)
3. **splash.png** - заставка при запуске приложения (200x200px)
4. **og-image.png** - изображение для социальных сетей (1200x630px)
5. **apple-icon.png** - иконка для Apple устройств (180x180px)
6. **icon-light-32x32.png** - маленькая иконка для светлой темы (32x32px)
7. **icon-dark-32x32.png** - маленькая иконка для темной темы (32x32px)

## Пошаговая инструкция для обновления на GitHub и Vercel

### ШАГ 1: Скачать обновленный код из v0

1. В этом чате нажмите на **три точки** (⋮) в правом верхнем углу
2. Выберите **"Download ZIP"**
3. Сохраните файл на компьютер
4. Распакуйте ZIP-архив

### ШАГ 2: Заменить файлы в вашем локальном проекте

Если у вас уже есть папка с проектом на компьютере:

1. Откройте папку с вашим проектом `statapp`
2. Скопируйте следующие файлы из скачанного архива:
   - Все файлы из папки `public/` (icon.png, icon-200x200.png, splash.png и т.д.)
   - Файл `app/.well-known/farcaster.json/route.ts`
   - Файл `app/layout.tsx`

### ШАГ 3: Загрузить изменения на GitHub

Откройте терминал/командную строку в папке с проектом и выполните:

\`\`\`bash
git add .
git commit -m "Update app icons with StatApp logo"
git push
\`\`\`

**Объяснение команд:**
- `git add .` - подготовить все измененные файлы к загрузке
- `git commit -m "..."` - создать точку сохранения с описанием
- `git push` - загрузить изменения на GitHub

### ШАГ 4: Автоматический деплой в Vercel

Vercel автоматически обнаружит изменения в GitHub и начнет новый деплой:

1. Откройте https://vercel.com
2. Найдите ваш проект `statapp`
3. Вы увидите статус "Building..." (идет сборка)
4. Подождите 1-2 минуты до статуса "Ready"

### ШАГ 5: Очистить кэш Farcaster (ВАЖНО!)

После деплоя нужно очистить кэш Farcaster, чтобы увидеть новую иконку:

1. Откройте https://warpcast.com/~/developers
2. Найдите ваше приложение StatApp
3. Нажмите **"Refresh"** или кнопку обновления
4. Подождите 1-2 минуты

### ШАГ 6: Проверить результат

1. Откройте ваше приложение в Farcaster
2. Теперь вместо привидения должен отображаться ваш логотип StatApp
3. При запуске приложения также должна показываться ваша заставка

## Если что-то пошло не так

### Проблема: Иконка не обновилась

**Решение:**
1. Подождите 5-10 минут (Farcaster кэширует изображения)
2. Попробуйте открыть приложение в режиме инкогнито браузера
3. Проверьте, что файлы действительно загрузились на GitHub
4. Убедитесь, что деплой в Vercel завершился успешно (зеленая галочка)

### Проблема: Изображения не отображаются (ошибка 404)

**Решение:**
1. Убедитесь, что все файлы находятся в папке `public/`
2. Проверьте, что имена файлов написаны правильно (без заглавных букв)
3. Сделайте повторный деплой в Vercel:
   - Откройте проект в Vercel
   - Вкладка "Deployments"
   - Три точки у последнего деплоя → "Redeploy"

### Проблема: Git команды не работают

**Решение:**
1. Убедитесь, что вы находитесь в правильной папке проекта
2. Проверьте, что Git установлен: `git --version`
3. Если Git не установлен, скачайте с https://git-scm.com/
